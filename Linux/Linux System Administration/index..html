Command Description Examples
chmod

change file mode bits
  • chmod u+w filename.extension (add write permission to user(owner) (u))
  • chmod g+r filename.extension (add read permission to group (g))
  • chmod o-r filename.extension (delete reading permission to other (o))
  • You can add / delete multi permissions like (chmod ugo+r filename.extension to add read permission to user,group and other)
  • for all you can use chmod a+x filename.extension this will give all Execute permission
  • using chmod -R will directories will change mode recursively
chown change file owner and group
  • sudo chown moataz site/ to change owner of site directory to be user => moataz
  • sudo chown :admins site/ to change owner of site directory to be group => admins
  • sudo chown moataz:admins site/ to change owner of site directory to be
    • user => moataz
    • group => admins
usermod modify a user account
  • usermod -G admins ola add user ola to group admins
